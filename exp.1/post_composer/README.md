# Postly — Social Content Studio

React implementation of Experiments 1.1.1 and 1.1.2: a responsive, multi-platform post composer with real-time platform validation and a persistent draft-management workspace.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The app includes platform-specific character, hashtag, and media validation; draft CRUD; scheduling; search; calendar and analytics views; `localStorage` persistence; memoized derived data; lazy-loaded analytics; responsive navigation; and dark mode.
